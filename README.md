# 北九州おでかけ情報まとめ

北九州市内+下関の一部の**催事・イベント**を、各公式サイトや地域メディアの公開情報から
自動で集めて1ページにまとめています。

👉 **https://hirofuminagatomi-create.github.io/kitaq-odekake/**

## お断り

- **開催情報は必ず公式サイトでご確認ください。** 個人が自動収集したものです。
  日程の変更・中止を保証できません。各イベントに出典リンクを付けています。
- 日程がはっきり書かれていない催しには「日程は目安」と表示しています。
- 映画の上映情報は扱っていません(劇場公式や映画情報サイトの方が確実なため)。

## 中身

- `docs/index.html` — ページ本体(1枚で完結。絞り込みもページ内で動きます)
- `docs/events.json` — 同じ内容のデータ

自動生成物です。手で編集しても次回の更新で上書きされます。
